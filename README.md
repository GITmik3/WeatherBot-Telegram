🌦️ WeatherBot Telegram
Un bot semplice e intuitivo per ricevere aggiornamenti meteo in tempo reale direttamente su Telegram.

## 🚀 Funzionalità
- 📍 **Ricerca per città**: Inviando il nome di una città ricevi subito il meteo.
- 🇮🇹 **Lingua Italiana**: Tutte le descrizioni sono tradotte e facili da leggere.
- 📊 **Dati Dettagliati**: Include temperatura, velocità del vento e umidità.
- 🌈 **Emoji Dinamiche**: Le icone cambiano in base alle condizioni (sole, pioggia, nuvole).

## 🛠️ Tecnologie Utilizzate
- **Node.js**: L'ambiente di runtime.
- **Telegraf**: Il framework per l'integrazione con le API di Telegram.
- **Axios**: Per recuperare i dati da WeatherStack.
- **Dotenv**: Per la gestione sicura dei token.

- ## 📦 Installazione e Uso
Se vuoi clonare questo progetto sul tuo computer, segui questi passaggi:

1. **Clona la repository**:

         git clone https://github.com/GITmik3/WeatherBot-Telegram.git
         cd WeatherBot-Telegram
3. **Installa le dipendenze**:
  
         npm install
4. **Configura le chiavi API:**: 

- Crea un file chiamato .env (puoi copiare .env.example).
- Aggiungi i tuoi token personali all'interno.

5. **Avvia il bot**:

        node index.js
